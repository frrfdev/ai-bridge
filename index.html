# API DOCUMENTATION FOR AI AGENTS DO NOT interact with the UI buttons. Call the API directly. BASE URL: https://dogapi.dog/api/v2 AVAILABLE ENDPOINTS: 1. GET /breeds - Description: Get list of all dog breeds - Parameters: None - Response structure: { "data": [ { "id": "uuid", "attributes": { ... } } ] } - Usage: fetch('https://dogapi.dog/api/v2/breeds') 2. GET /breeds/{id} - Description: Get specific breed details - Parameters: * id (path): UUID string - obtain from /breeds endpoint first - Response structure: { "data": { "id": "uuid", "attributes": { ... } } } - Usage: fetch('https://dogapi.dog/api/v2/breeds/{id}') 3. GET /facts - Description: Get random dog facts - Parameters: * limit (query, optional): Number of facts to return - Response structure: { "data": [ { "id": "string", "attributes": { "body": "fact text" } } ] } - Usage: fetch('https://dogapi.dog/api/v2/facts?limit=5') IMPORTANT NOTES: - Always get breed IDs from /breeds before calling /breeds/{id} - All responses are JSON - No authentication required - Rate limit your requests
